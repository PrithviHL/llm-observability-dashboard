# llm-observability-dashboard
Self-hosted LLM observability &amp; eval dashboard — ingests OpenTelemetry GenAI spans from OpenAI, Anthropic, Google, LangChain, LlamaIndex, and vLLM; runs DeepEval/RAGAS eval jobs; detects prompt drift via PSI; and alerts via Prometheus → Slack/PagerDuty. Built with ClickHouse, Postgres, and Next.js 15.
